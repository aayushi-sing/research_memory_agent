🧠 Persistent Research Memory Agent

An agentic AI system that builds long-term semantic memory across research papers, articles, and notes.

Unlike traditional PDF chat or RAG apps, this project focuses on:

-persistent memory

-cross-document reasoning

-contradiction detection

-concept relationship mapping

-evolving research intelligence

The agent remembers uploaded knowledge over time and connects ideas across multiple sources to generate deeper research insights.

✨ Features

  - Persistent semantic memory using vector databases

  - Cross-document reasoning and synthesis

  - Contradiction detection between research papers

  - Interactive knowledge graph visualization

  - Timeline-based memory tracking

  - Multiple research modes:

      -Answer Questions
      -Connect Ideas
      -Contradiction Analysis
  
💡 Example Queries

Answer

  -What is the main contribution of these papers?
  
  -What are the key limitations discussed across the papers?
  
Connect Ideas

  -What recurring themes appear across all uploaded papers?
  
  -How are retrieval systems connected to long-term memory architectures?

  
Contradict

  -Which papers disagree about handling long-term memory?
  
  -What contradictions exist between retrieval-based memory and long-context approaches?
